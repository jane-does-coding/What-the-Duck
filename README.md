## Setup

### 1. Clone the repo

```bash
git clone https://github.com/jane-does-coding/What-the-Duck
cd what-the-duck
```

### 2. Install the dependencies

```bash
npm i
```

### 3. Configure .env

```bash
DATABASE_URL="your mongodb"
NEXTAUTH_SECRET="random-secret"
GEMINI_API_KEY="gemini api key"
```

### 4. Setup Prisma

```bash
npx prisma generate
```

### 4. Run the project

```bash
npm run dev
```
